About Cyber Sentinel

Cyber Sentinel is a senior project by Abdullah Mehtab and Nabeel Mahmood at Forman Christian College, designed to protect small networks from cyber threats.

Built with Wazuh and the ELK Stack on a Raspberry Pi 5, it offers an accessible solution for small businesses and homes, detecting threats with over 90% accuracy in tests.

Key Features

Real-Time Monitoring

Detects threats like brute-force attacks and network scans instantly, keeping your network safe 24/7.

Advanced Analytics

Visualize security data with Kibana dashboards for actionable insights into threats and trends.

Smart Alerts

Receive customizable, detailed email notifications for critical events, formatted for clarity.

How It Works

Log Collection

Wazuh agents on endpoints gather system logs and send them to the Raspberry Pi.

Threat Detection

Wazuh Manager analyzes logs using rules to identify threats like scans or login attempts.

Data Processing

Filebeat and Logstash process alerts, sending them to Elasticsearch and formatting emails.

Visualization

Kibana displays real-time security insights through an intuitive dashboard.

Frequently Asked Questions

Cyber Sentinel is an open-source security monitoring system for small businesses and home networks, using Wazuh and the ELK Stack on a Raspberry Pi 5 to detect threats in real time.

Requires a Raspberry Pi 5 with 8GB RAM running Kali Linux or Ubuntu Server. Endpoints need Wazuh agents and can run Windows or Linux.

Detects aggressive scans (e.g., Nmap), brute-force attacks (e.g., Hydra), exploits (e.g., EternalBlue), file changes, and more, with over 90% accuracy in tests.

Download the installer, run it on your Raspberry Pi, and follow the wizard. Agents are then installed on endpoints. Setup takes under 30 minutes.

Optimized for small to medium networks. Larger setups may require additional hardware for optimal performance.
