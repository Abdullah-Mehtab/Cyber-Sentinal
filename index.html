Frequently Asked Questions

What is Cyber Sentinel?

Cyber Sentinel is a cost-effective security monitoring system for small networks, using Wazuh and ELK Stack on a Raspberry Pi to detect threats and provide insights.

How does it work?

Wazuh agents send logs to the Raspberry Pi, where Wazuh Manager analyzes them. Alerts are stored in Elasticsearch, formatted by Logstash, and visualized in Kibana.

What are the system requirements?

A Raspberry Pi 5 with 8GB RAM running Kali Linux or Ubuntu Server, plus Windows or Linux endpoints with Wazuh agents.

How do I install it?

Download the Linux installer, run it on your Raspberry Pi, install Wazuh agents on devices, and connect them to the Manager.

What threats can it detect?

It detects scans, brute-force logins, DDoS attempts, and more, with over 90% accuracy in tests.

Is it free?

Yes, it's open-source using free tools; the only cost is the Raspberry Pi hardware.
