Frequently Asked Questions

What is Cyber Sentinel?

Cyber Sentinel is an integrated security monitoring system designed for small businesses and home networks. It provides real-time threat detection, advanced analytics, and instant alerts for potential security breaches.

What are the system requirements?

The system requires a Raspberry Pi 5 with at least 8GB RAM, running Kali Linux or Ubuntu Server. The monitored endpoints can be Windows or Linux machines with the Wazuh agent installed.

How does the alert system work?

Cyber Sentinel uses a combination of Wazuh and the ELK Stack to monitor system logs and generate alerts. Critical alerts are sent via email with detailed HTML formatting, and all events can be viewed through the Kibana dashboard.

Is it suitable for large networks?

Currently, Cyber Sentinel is optimized for small to medium-sized networks. While it can handle multiple endpoints, it's best suited for small businesses and home networks due to the Raspberry Pi's resource constraints.

What types of threats can it detect?

The system can detect various threats including brute force attacks, unauthorized access attempts, suspicious network scans, and file integrity changes. It's particularly effective at detecting traditional cyber attacks and system-level security events.

How do I get started?

Simply download our Linux installer, follow the installation guide, and set up the Wazuh agents on your endpoints. The system includes a user-friendly GUI for monitoring and management.
